### 特 性

- 支持多种内置的数据结构
- 支持磁盘数据存储和垃圾回收
- 支持 HTTP 协议的数据操作协议
- 支持磁盘数据静态加密和数据压缩
- 支持服务数据访问 IP 白名单功能
- 高吞吐量、低延迟批量数据写入效率
